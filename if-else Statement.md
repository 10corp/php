## PHP-তে if/else Statements: ভূমিকা

### if-else স্টেটমেন্ট কী?

PHP-তে `if-else` স্টেটমেন্ট হলো শর্তানুযায়ী কোড নিয়ন্ত্রণ করার একটি উপায়। এটি প্রোগ্রামকে সিদ্ধান্ত নিতে সাহায্যে করে: যদি একটি শর্ত সত্য হয়, তবে নির্দিষ্ট কোড চলে; অন্যথায় অন্য কোড চলে। আপনার `eat_out.php` কোডে আপনি ইতিমধ্যে `if-else` ব্যবহার করেছেন, যেমন:

```php
if ($money > 300 || $friend_pays === true) {
    echo "খেতে যেতে পারবে!";
} else {
    echo "ঘরে থাকো।";
}
```

### কেন if-else শিখবেন?

*   **সিদ্ধান্ত গ্রহণ**: আপনার প্রোগ্রামে শর্তের উপর ভিত্তি করে বিভিন্ন ফলাফল তৈরি করতে।
*   **জটিল শর্ত**: আপনার শেখা লজিক্যাল অপারেটর (`&&`, `||`, `!`) এবং ইন্টিজার (`$money`) ব্যবহার করে।
*   **প্রেক্ষাপটে**: `eat_out.php` এ `$money > 300` বা `$friend_pays` চেক করার জন্য `if-else` দরকার।

### প্রকারভেদ

1.  **সাধারণ if**: শুধু একটি শর্ত চেক করে।
2.  **if-else**: শর্ত সত্য বা মিথ্যার জন্য দুটি ফলাফল।
3.  **if-elseif-else**: একাধিক শর্ত চেক করে।
4.  **নেস্টেড if**: একটি `if` এর ভেতর আরেকটি `if`।

## if/else স্টেটমেন্টের বিস্তারিত ব্যাখ্যা

নিচে প্রতিটি ধরনের `if-else` স্টেটমেন্টের ব্যাখ্যা, সিনট্যাক্স, উদাহরণ, এবং আপনার `eat_out.php` প্রেক্ষাপটে ব্যবহার দেওয়া হলো।

### 1\. সাধারণ if স্টেটমেন্ট

*   **ব্যাখ্যা**: শুধুমাত্র একটি শর্ত চেক করে। শর্ত সত্য হলে কোড চলে, মিথ্যা হলে কিছু হয় না।
*   **সিনট্যাক্স**:
    
    ```php
    if (শর্ত) {
        // কোড যদি শর্ত সত্য হয়
    }
    ```
    
*   **উদাহরণ**:
    
    ```php
    $age = 20;
    if ($age >= 18) {
        echo "প্রাপ্তবয়স্ক!\n";
    }
    ```
    
    *   ফলাফল: `প্রাপ্তবয়স্ক!`
*   **eat\_out.php প্রেক্ষাপট**:
    
    ```php
    $money = 400;
    if ($money > 300) {
        echo "খেতে যেতে পারবে!\n";
    }
    ```
    
    *   ফলাফল: `খেতে যেতে পারবে!`

### 2\. if-else স্টেটমেন্ট

*   **ব্যাখ্যা**: শর্ত সত্য হলে একটি কোড চলে, মিথ্যা হলে অন্য কোড চলে।
*   **সিনট্যাক্স**:
    
    ```php
    if (শর্ত) {
        // কোড যদি শর্ত সত্য হয়
    } else {
        // কোড যদি শর্ত মিথ্যা হয়
    }
    ```
    
*   **উদাহরণ**:
    
    ```php
    $temperature = 25;
    if ($temperature > 30) {
        echo "গরম আছে, ফ্যান চালাও!\n";
    } else {
        echo "আবহাওয়া ঠিক আছে।\n";
    }
    ```
    
    *   ফলাফল: `আবহাওয়া ঠিক আছে।`
*   **eat\_out.php প্রেক্ষাপট**:
    
    ```php
    $money = 400;
    $friend_pays = false;
    if ($money > 300 || $friend_pays === true) {
        echo "বাইরে খেতে যাও!\n";
    } else {
        echo "ঘরে থাকো।\n";
    }
    ```
    
    *   ফলাফল: `বাইরে খেতে যাও!` (কারণ `$money > 300` সত্য)

### 3\. if-elseif-else স্টেটমেন্ট

*   **ব্যাখ্যা**: একাধিক শর্ত চেক করে। প্রথম সত্য শর্তের কোড চলে, অথবা শেষে `else` চলে।
*   **সিনট্যাক্স**:
    
    ```php
    if (শর্ত১) {
        // কোড যদি শর্ত১ সত্য হয়
    } elseif (শর্ত২) {
        // কোড যদি শর্ত২ সত্য হয়
    } else {
        // কোড যদি কোনো শর্ত সত্য না হয়
    }
    ```
    
*   **উদাহরণ**:
    
    ```php
    $score = 85;
    if ($score >= 90) {
        echo "গ্রেড: A+\n";
    } elseif ($score >= 80) {
        echo "গ্রেড: A\n";
    } elseif ($score >= 70) {
        echo "গ্রেড: B\n";
    } else {
        echo "গ্রেড: F\n";
    }
    ```
    
    *   ফলাফল: `গ্রেড: A`
*   **eat\_out.php প্রেক্ষাপট**:
    
    ```php
    $money = 400;
    $people = 3;
    if ($money > 500) {
        echo "রেস্টুরেন্টে খেতে যাও!\n";
    } elseif ($money > 300 && $people >= 2) {
        echo "পিকনিকে খেতে যাও!\n";
    } else {
        echo "ঘরে খাও।\n";
    }
    ```
    
    *   ফলাফল: `পিকনিকে খেতে যাও!`

### 4\. নেস্টেড if স্টেটমেন্ট

*   **ব্যাখ্যা**: একটি `if` এর ভেতর আরেকটি `if` ব্যবহার। জটিল শর্ত চেক করতে ব্যবহৃত।
*   **সিনট্যাক্স**:
    
    ```php
    if (শর্ত১) {
        if (শর্ত২) {
            // কোড যদি উভয় শর্ত সত্য হয়
        }
    }
    ```
    
*   **উদাহরণ**:
    
    ```php
    $has_ticket = true;
    $is_adult = true;
    if ($has_ticket) {
        if ($is_adult) {
            echo "মুভি দেখতে পারবে!\n";
        } else {
            echo "শুধুমাত্র প্রাপ্তবয়স্কদের জন্য!\n";
        }
    } else {
        echo "টিকিট নেই।\n";
    }
    ```
    
    *   ফলাফল: `মুভি দেখতে পারবে!`
*   **eat\_out.php প্রেক্ষাপট**:
    
    ```php
    $money = 400;
    $friend_pays = true;
    if ($money > 300 || $friend_pays) {
        if ($people >= 2) {
            echo "গ্রুপে খেতে যাও!\n";
        } else {
            echo "একা খেতে যাও!\n";
        }
    } else {
        echo "ঘরে থাকো।\n";
    }
    ```
    
    *   ফলাফল: `গ্রুপে খেতে যাও!`

## টার্নারি অপারেশনের সাথে তুলনা

আপনি টার্নারি অপারেশন শিখেছেন। `if-else` এবং টার্নারি তুলনা:

*   **if-else**:
    
    ```php
    if ($money > 300) {
        $message = "খেতে যাও!";
    } else {
        $message = "ঘরে থাকো।";
    }
    ```
    
*   **টার্নারি**:
    
    ```php
    $message = ($money > 300) ? "খেতে যাও!" : "ঘরে থাকো।";
    ```
    
*   **কখন ব্যবহার করবেন**: সাধারণ শর্তে টার্নারি সংক্ষিপ্ত, কিন্তু জটিল বা একাধিক কোড ব্লকের জন্য `if-else` ভালো।

## সম্পূর্ণ কোড উদাহরণ

নিচে একটি সম্পূর্ণ কোড দেওয়া হলো, যা সব ধরনের `if-else` স্টেটমেন্ট (`if`, `if-else`, `if-elseif-else`, নেস্টেড) কভার করে, লজিক্যাল অপারেটর (`&&`, `||`), টার্নারি, এবং আপনার `eat_out.php` প্রেক্ষাপটে (`$money`, `$friend_pays`, `$activity`) সংযুক্ত করে। ফাইলটি `D:\laragon\www\php\conditions_practice\if_else_practice` ফোল্ডারে `if_else.php` নামে সংরক্ষণ করুন।

// ভ্যারিয়েবল ঘোষণা  
$money = 400;  
$friend\_pays = true;  
$people = 3;  
$ticket\_price = 100;  
$activity = "খাওয়া,দেখা";

// ১. সাধারণ if  
if ($money > 300) {  
echo "সাধারণ if: খেতে যেতে পারবে!\\n";  
}

// ২. if-else  
if ($money > 300 || $friend\_pays === true) {  
echo "if-else: বাইরে খেতে যাও!\\n";  
} else {  
echo "if-else: ঘরে থাকো।\\n";  
}

// ৩. if-elseif-else  
if ($money > 500) {  
echo "if-elseif-else: রেস্টুরেন্টে খেতে যাও!\\n";  
} elseif ($money > 300 && $people >= 2) {  
echo "if-elseif-else: পিকনিকে খেতে যাও!\\n";  
} else {  
echo "if-elseif-else: ঘরে খাও।\\n";  
}

// ৪. নেস্টেড if  
if ($money > 300 || $friend\_pays) {  
if ($people >= 2) {  
echo "নেস্টেড if: গ্রুপে খেতে যাও!\\n";  
} else {  
echo "নেস্টেড if: একা খেতে যাও!\\n";  
}  
} else {  
echo "নেস্টেড if: ঘরে থাকো।\\n";  
}

// টিকিট চেকার (পূর্বের কোডের সাথে)  
$tickets = intdiv($money, $ticket\_price);  
if ($tickets >= 2 && $money % $ticket\_price > 50) {  
echo "টিকিট চেকার: কমপক্ষে $tickets টি টিকিট কিনতে পারবে!\\n";  
} else {  
echo "টিকিট চেকার: টিকিট বা অবশিষ্ট টাকা কম।\\n";  
}

// টার্নারি দিয়ে তুলনা  
$plan = ($money > 300 && $friend\_pays) ? str\_replace("খাওয়া", "খেতে", $activity) : "কোনো অ্যাকটিভিটি নেই।";  
echo "টার্নারি: $plan\\n";

?>

## কোড ব্যাখ্যা

### প্রধান উপাদান

*   **if-else প্রকার**: সাধারণ `if`, `if-else`, `if-elseif-else`, এবং নেস্টেড `if` ব্যবহার দেখানো হয়েছে।
*   **জটিল শর্ত**: `$money > 300 || $friend_pays`, `$money > 300 && $people >= 2` ইত্যাদি লজিক্যাল অপারেটর ব্যবহার।
*   **ইন্টিজার**: `intdiv()` এবং `%` দিয়ে টিকিট গণনা, আপনার “Numbers, Part 1: Integers” শিক্ষার সাথে সংযুক্ত।
*   **স্ট্রিং ফাংশন**: `str_replace()` দিয়ে `eat_out.php` এর মতো অ্যাকটিভিটি হ্যান্ডেল।
*   **টার্নারি**: টার্নারি অপারেশন দিয়ে `if-else` এর তুলনা।
*   **UTF-8**: বাংলা টেক্সটের জন্য `header()` ব্যবহৃত।

### আউটপুট

টার্মিনালে চালালে:

```css
সাধারণ if: খেতে যেতে পারবে!
if-else: বাইরে খেতে যাও!
if-elseif-else: পিকনিকে খেতে যাও!
নেস্টেড if: গ্রুপে খেতে যাও!
টিকিট চেকার: টিকিট বা অবশিষ্ট টাকা কম।
টার্নারি: খেতে,দেখা
```

**নোট**: টিকিট চেকারে `$money % $ticket_price = 0`, তাই `> 50` শর্ত মিথ্যা, ফলে “টিকিট বা অবশিষ্ট টাকা কম।” আউটপুট হয়।

## গুরুত্বপূর্ণ টিপস

1.  **শর্তের অগ্রাধিকার**:
    *   জটিল শর্তে বন্ধনী `()` ব্যবহার করুন:
        
        ```php
        if (($money > 300 || $friend_pays) && $people >= 2) { ... }
        ```
        
2.  **নেস্টেড if এড়ানো**:
    *   অতিরিক্ত নেস্টিং কোড জটিল করে। একত্রিত শর্ত ব্যবহার করুন:
        
        ```php
        if ($money > 300 && $people >= 2) { ... }
        ```
        
3.  **ডিবাগিং**:
    *   শর্তের ফলাফল চেক করুন:
        
        ```php
        var_dump($money > 300 || $friend_pays);
        echo "Tickets: $tickets, Remaining: " . ($money % $ticket_price) . "\n";
        ```
        
4.  **UTF-8**:
    *   বাংলা টেক্সটের জন্য ফাইল UTF-8 এ সংরক্ষণ করুন।
    *   `header('Content-Type: text/html; charset=UTF-8');` ব্যবহার করুন।

## আপনার প্রেক্ষাপটে সংযোগ

*   **eat\_out.php**: আপনার পূর্বের কোডে `$money > 300 || $friend_pays` এর মতো `if-else` ছিল। এই কোডে একই লজিক বিভিন্ন `if-else` প্রকারে প্রয়োগ করা হয়েছে।
*   **ইন্টিজার**: `intdiv()` এবং `%` আপনার “Numbers, Part 1: Integers” শিক্ষার সাথে সম্পর্কিত।

*   **কাস্টম ফাংশন**: `if-else` ব্যবহার করে ফাংশন তৈরি করুন:
    
    ```php
    function getPlan($money, $people) {
        if ($money > 500) {
            return "রেস্টুরেন্ট";
        } elseif ($money > 300 && $people >= 2) {
            return "পিকনিক";
        } else {
            return "ঘরে খাও";
        }
    }
    ```

    ```php
        }
    }
    ```
